## Gemini Added Memories (audio_mixdown project)
- React 및 TypeScript를 사용한다.
- CSS는 Tailwind CSS를 기본으로 사용한다.
- 모든 데이터 호출은 API를 통해서만 호출한다.
- 정적인 CSS 정의는 항상 CSS 파일 내에서만 정의하고, JavaScript 파일에서는 절대 정의하지 않는다.
- React 애플리케이션에서 `public/index.html` 파일은 유일한 정적 HTML 진입점이며, React 컴포넌트 내의 JSX는 동적인 UI를 구성하므로 별도의 정적 HTML 파일로 분리하지 않는다.
- 이 프로젝트는 front-end 영역이며, back-end 영역은 PHP로 별도 구성될 예정이다.
- back-end 영역의 모든 파일은 ./api 에 위치 시킨다. 
- 프로젝트의 기본 디렉토리는 /Users/norton/Ai_project/audio_mixdown 입니다.
- 프로젝트 요약 및 디렉토리 구조 보고 시 `node_modules` 디렉토리는 항상 제외한다.
